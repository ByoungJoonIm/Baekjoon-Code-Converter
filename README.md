# 사용법(GUI)
0. 내장 jar를 사용하므로 bcConverter.exe 이외의 파일은 모두 지워도 상관 없다.
1. bcConverter.exe를 실행한다.(java 기반의 jsmooth wrapper를 사용하므로 JRE 필요)
2. IDE에서 백준에 올릴 코드를 복사한다.
3. 프로그램의 convert를 클릭한다.(클립보드에서 package명 삭제 및 class명을 Main으로 변경)
4. 백준에 붙여 넣는다.
